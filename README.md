# SUPERMARKET SALES DATA ANALYSIS

## Project Objective
The supermarket wants to create a sales report from three different stores in three cities to know about the sales and to implement more ideas to grow sales.

## Dataset used
- <a href = "https://github.com/Anna-Vincent/Data-analysis-dashboard/blob/main/Excel%20Dashboard.xlsx" > Dataset</a>

## To Resolve these Questions
- Which city has highest sales?
- Which product has the highest sales?
- Who purchased more -  Men or Women?
- To know the preferred payment type
- To identify trend over time.
- Relationship between product unit price and category.

-Dashboard Interaction <a herf="https://github.com/Anna-Vincent/Data-analysis-dashboard/blob/main/Screenshot%202025-12-04%20194401.png"> View Dashboard</a>

## Process
- Data is been preprocessed to remove anomalities.
- To make sure Data is consistent and clean.
- Create pivot tables according to the questions asked,
- Create a dashboard by  merging all the pivot tables and adding slicer to it.
